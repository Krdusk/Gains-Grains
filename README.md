# README: G&G Gains and Grains

## ## Project Overview
**G&G Gains and Grains** is an e-commerce platform developed for the Database Systems course. The project demonstrates a functional integration between a web interface and a relational database, allowing for dynamic product management and data persistence.

## ## Design Inspiration
The user interface of G&G Gains and Grains features a unique visual identity. The aesthetic was inspired by:
* **Neon Aesthetics:** High-contrast elements and vibrant glow effects to give the site a modern, high-energy feel.
* **Pixar Animations:** Inspiration drawn from the clean, expressive, and friendly character of Pixar’s visual storytelling to ensure the UI remains approachable and engaging.

## ## Technical Stack
* **Server Environment:** XAMPP
* **Backend:** PHP
* **Database:** MySQL (`gains_grains_db`)
* **Data Format:** JSON
* **Frontend:** HTML, CSS, JavaScript

---

## ## Prerequisites
To run this project, you must have **XAMPP** installed on your local machine.

### ### How to Download and Install XAMPP
1.  Go to the official [Apache Friends website](https://www.apachefriends.org/download.html).
2.  Select the version compatible with your operating system.
3.  Download and run the installer, following the setup wizard (ensure Apache and MySQL are selected).
4.  Complete the installation in your preferred directory (e.g., `C:\xampp`).

---

## ## How to Run the Project in XAMPP

### ### 1. Prepare the Project Files
* Locate your XAMPP installation folder and open the `htdocs` folder.
* Create a folder named `gains-and-grains` and paste all project files inside.

### ### 2. Start the Servers
* Open the **XAMPP Control Panel**.
* Click **Start** for both **Apache** and **MySQL**.

### ### 3. Setup the Database
* Open your browser and go to `http://localhost/phpmyadmin`.
* Create a new database named `gains_grains_db`.
* Use the **Import** tab to upload the `.sql` file included in this project.

### ### 4. Access the Website
* Type `http://localhost/gains-and-grains` in your browser to view the live site.

---

> **Connection Note:** The system is configured to connect to the database using default XAMPP credentials (User: `root`, Password: `""`).
